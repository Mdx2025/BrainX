# BrainX V6 Scripts Manifest

BrainX scripts are maintenance utilities for memory capture, curation, import, cleanup, evaluation, and backup.

## Common Groups

- Capture: `memory-distiller.js`, `fact-extractor.js`, `session-harvester.js`, `memory-md-harvester.js`
- Import/sync: `memory-bridge.js`, `import-workspace-memory-md.js`, `import-knowledge-md.js`, `knowledge-sync.js`
- Quality: `quality-scorer.js`, `cleanup-low-signal.js`, `dedup-supersede.js`, `contradiction-detector.js`
- Governance: `auto-promoter.js`, `promotion-applier.js`, `cleanup-promotion-suggestions.js`
- Knowledge: `knowledge-locate.js`, `new-knowledge-topic.js`, `seed-knowledge-library.js`
- Evaluation: `eval-memory-quality.js`, `generate-eval-dataset-from-memories.js`
- Backup: `backup-brainx.sh`, `restore-brainx.sh`, `weekly-backup.sh`

Run scripts from the repository root unless a script documents otherwise.
